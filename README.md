# NextGen Transportation Management System (TMS)

Domain: Logistics | Category: Transportation Management Systems (TMS) | Products Analyzed: MercuryGate.

## Stack

- Next.js 15 App Router
- TypeScript
- TailwindCSS
- Supabase Auth, PostgreSQL, Realtime, Storage, Edge Functions

## Core Modules

- Authentication and role-based access control
- Customers, orders, shipments, carriers, rates, and routes
- Shipment tracking and document upload
- Reporting dashboard and customer portal
